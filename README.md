# AutoCalculator v1.0.0 (통합 미세조직 및 자동화 분석 시스템)

재료 공학 연구 및 품질 분석을 위한 **범용 미세조직 자동 분석 툴**입니다. 본 프로그램은 EBSD 데이터 분석, XRD 분석, 그리고 이미지 기반 두께 자동 측정 등 다양한 기능을 하나의 대시보드에서 제공합니다.

## 🚀 통합 솔루션 개요

AutoCalculator는 단순한 데이터 계산기를 넘어, 실험 데이터(XRD, EBSD)와 이미지 데이터(Microstructure)를 결합하여 정량적 분석 결과를 도출하는 통합 시스템입니다.

## ✨ 핵심 모듈 및 기능

### 1. XRD & 미세조직 분석 (Microstructure Module)
- **XRD 데이터 처리**: XRD 피크 데이터를 분석하여 결정립 크기(Grain size)를 자동 산출합니다.
- **상 분석 (Phase Analysis)**: TC(Texture Coefficient) 및 C/N Ratio(질화물/탄화물 비율) 등 재료 배향성을 계산합니다.
- **정량적 평가**: 재료의 물리적 특성을 수치화하여 그래프 및 리포트로 제공합니다.

### 2. 코팅 두께 자동 측정 (Auto-Thickness Module)
- **에지 검출 알고리즘 (Edge Detection)**: 이미지 처리 알고리즘을 통해 다층 코팅(Multi-layer)의 경계를 자동으로 인식합니다.
- **자동 두께 산출**: 수동 측정의 오차를 줄이고, 이미지 내 여러 지점의 두께와 편차를 자동으로 계산합니다.
- **실시간 캔버스 분석**: 분석된 레이어를 캔버스 위에서 시각적으로 확인하고 수정할 수 있습니다.

### 3. 고정밀 EBSD 분석 (EBSD Module)
- **결정립 분석**: EBSD(.ang) 파일의 결정립 크기를 면적 가중치 기반(D10, D50, D90)으로 정밀 산출합니다.
- **상 분율 측정**: 상(Phase) 간의 결합성 및 분율을 고해상도 이미지 파이프라인으로 분석합니다.
- **특수 경계 검출**: Sigma-2 등 Coincidence Site Lattice(CSL) 경계 특성을 정량화합니다.

## 🛠 기술적 특징

- **통합 대시보드**: React와 TypeScript 기반의 현대적인 UI로 모든 분석 모듈을 직관적으로 제어합니다.
- **고성능 연산 파이프라인**: Python(NumPy, SciPy) 기반의 강력한 분석 코어를 활용하여 대용량 데이터를 빠르게 처리합니다.
- **플렉서블 실행 환경**: 전용 런처(`START_HERE.bat`)를 통해 복잡한 환경 설정 없이 실행할 수 있는 포터블 환경을 제공합니다.

## 📦 시작하기 (설치 및 실행)

1.  **리포지토리 클론 (또는 다운로드)**:
    ```bash
    git clone https://github.com/shin9602/Microstructure-Analyzer.git
    cd Microstructure-Analyzer
    ```
2.  **프로그램 실행**:
    - `START_HERE.bat` 파일을 더블 클릭하세요.
    - 실행에 필요한 모든 환경(Node.js, Python 라이브러리)이 자동으로 구축되고 프로그램이 실행됩니다.
